# UI设计文档：资源模块与资源中心视图

## 1. 设计原则

### 1.1 用户体验原则
- **一致性**：保持与现有界面一致的设计语言
- **直观性**：信息层次清晰，操作路径明确
- **高效性**：减少操作步骤，提升工作效率
- **反馈性**：及时反馈操作结果，状态清晰可见

### 1.2 视觉设计原则
- **简洁**：避免信息过载，突出核心内容
- **层次**：合理使用颜色、字体、间距建立视觉层次
- **对比**：通过对比突出重要信息和状态
- **统一**：遵循 Ant Design 设计规范

## 3. 组件设计

### 3.1 分组树形组件 (GroupTree)
```
组件: GroupTree
位置: GroupManagement/GroupList.tsx
功能:
  - 树形展示分组结构
  - 支持展开/折叠
  - 支持拖拽排序
  - 点击选中分组
  - 显示分组资源数量

样式:
  - 使用 Ant Design Tree 组件
  - 自定义图标: 文件夹图标
  - 节点颜色: 选中时高亮
  - 右键菜单: 编辑、删除、添加子分组
```

### 3.2 分组详情组件 (GroupDetail)
```
组件: GroupDetail
位置: GroupManagement/GroupDetail.tsx

布局:

  - 中部: 资源列表表格
  - 底部: 批量操作面板
```

## 4. 交互流程设计

### 4.1 创建分组流程
```
1. 用户点击"创建分组"按钮
2. 弹出创建分组模态框
3. 填写表单:
   - 分组名称 (必填)
   - 描述 (可选)
   - 父分组 (可选，默认无)
   - 标签 (可选)
4. 点击"确定"
5. 后端创建分组
6. 前端刷新模块列表
7. 显示成功提示
```

### 4.2 添加资源到分组流程
```
1. 在分组详情页点击"添加资源"
2. 弹出资源选择器模态框
3. 显示未分组资源列表
4. 选择资源 (支持多选)
5. 点击"确定"
6. 后端更新资源模块归属
7. 前端刷新分组成员列表
8. 显示成功提示
```

### 4.3 批量应用规则流程
```
1. 在分组详情页点击"批量应用规则"
2. 选择规则类型 (流控/熔断)
3. 填写规则参数
4. 选择目标资源 (默认全选)
5. 点击"应用"
6. 后端批量创建规则
7. 前端显示进度条
8. 完成后显示结果摘要
```

### 4.4 资源详情页访问流程
```
1. 在列表页面，点击资源名称
2. 跳转到 /web/resources/{resource-name}
3. 页面加载:
   - 加载资源元数据
   - 加载
   - 加载
4. 显示资源详情页
5. 用户可以:
   - 查看规则状态
   - 快速编辑规则
   - 添加新规则
   - 切换规则状态
```

### 4.5 规则快速编辑流程
```
1. 在资源详情页规则列表点击"编辑"
2. 弹出编辑模态框
3. 表单预填当前规则参数
4. 修改参数
5. 点击"确定"
6. 后端更新规则
7. 前端刷新规则列表和状态卡片
8. 显示成功提示
```

## 5. 视觉设计规范

### 5.1 颜色规范
```
主色调:
  - 主色: #1890ff (Ant Design 蓝色)
  - 成功色: #52c41a (绿色)
  - 警告色: #faad14 (黄色)
  - 错误色: #f5222d (红色)
  - 禁用色: #d9d9d9 (灰色)

状态颜色:
  - 正常: #52c41a (绿色)
  - 警告: #faad14 (黄色)
  - 触发: #f5222d (红色)
  - 禁用: #d9d9d9 (灰色)
```

### 5.2 字体规范
```
标题:
  - 页面标题: 20px, 加粗
  - 分组标题: 16px, 加粗
  - 资源名称: 18px, 加粗

正文:
  - 正文: 14px, 常规
  - 小字: 12px, 常规

颜色:
  - 主要文字: rgba(0, 0, 0, 0.85)
  - 次要文字: rgba(0, 0, 0, 0.45)
  - 禁用文字: rgba(0, 0, 0, 0.25)
```

### 5.3 间距规范
```
页面间距:
  - 页面内边距: 24px
  - 卡片内边距: 16px
  - 元素间距: 8px, 16px, 24px

组件间距:
  - 表单行间距: 16px
  - 按钮间距: 8px
  - 表格行高: 48px
```

### 5.4 图标规范
```
分组图标:
  - 默认: 文件夹图标 (Ant Design FolderOutlined)
  - 展开: 文件夹打开图标
  - 折叠: 文件夹关闭图标

状态图标:
  - 正常: 对勾图标 (CheckCircleOutlined)
  - 警告: 警告图标 (WarningOutlined)
  - 触发: 错误图标 (CloseCircleOutlined)
  - 禁用: 禁用图标 (MinusCircleOutlined)
```

## 6. 响应式设计

### 6.1 断点设计
```
桌面端: >= 1200px
  - 模块管理: 左侧面板 40%，右侧面板 60%
  - 资源详情: 双列布局

平板端: 768px - 1199px
  - 模块管理: 左侧面板 30%，右侧面板 70%
  - 资源详情: 单列布局

移动端: < 768px
  - 模块管理: 单列布局，可折叠
  - 资源详情: 单列布局
```

### 6.2 适配策略
```
1. 使用 Ant Design 的 Grid 系统
2. 关键组件支持折叠/展开
3. 表格支持横向滚动
4. 表单使用响应式布局
```

## 7. 动画设计

### 7.1 过渡动画
```
页面切换:
  - 淡入淡出: 0.3s ease-in-out
  - 滑动: 0.3s ease-out

组件交互:
  - 模态框: 弹出 0.2s ease-out
  - 下拉菜单: 展开 0.15s ease-out
  - 树形展开: 展开 0.2s ease-out
```

### 7.2 状态变化动画
```
状态指示器:
  - 颜色变化: 0.3s ease-in-out
  - 图标变化: 0.2s ease-in-out

规则状态:
  - 启用/禁用: 滑动开关 0.2s ease-out
  - 触发状态: 闪烁动画 0.5s ease-in-out
```
