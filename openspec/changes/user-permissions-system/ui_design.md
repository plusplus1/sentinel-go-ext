## 一、核心页面（仅保留必要的）

### 1. 登录页面 (`/login`)
- 邮箱 + 密码登录
- 飞书 SSO 登录（可选）
- 未登录自动跳转此页

### 2. 模块管理 (`/modules`) — 所有角色可见
- 模块列表（名称、描述、成员数）
- 创建/编辑/删除模块
- 成员管理（添加/移除资源）

### 3. 资源中心 (`/resources`) — 所有角色可见
- 资源列表（名称、模块、规则数）
- 规则查看/编辑/切换
- 变更模块
- 添加/删除资源

### 4. 管理中心 (`/admin`) — 仅超级管理员
- 业务线管理（创建/编辑/删除）
- 成员管理（邀请/设置角色）
- App 管理（创建/编辑/授权）

---

## 二、权限控制

### 超级管理员 (`super_admin`)
- 侧边栏显示：资源中心、模块管理、管理中心
- 可以管理所有业务线和成员

### 业务线管理员 (`line_admin`)
- 侧边栏显示：资源中心、模块管理、管理中心（仅本线）
- 可以管理本线下的 App 和成员

### 普通成员 (`member`)
- 侧边栏显示：资源中心、模块管理
- 只能操作被授权的模块和资源
